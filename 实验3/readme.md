## 实验原理
贪心算法的基本思想
贪心算法求解优化问题的基本思想是：采用逐步构造最优解的方法。在每个
阶段，都作出一个当前最优的决策（在一定的标准下）。决策一旦作出，就不可
再更改（作出贪心决策的依据称为贪心准则）。贪心算法的一般步骤如下：
(1) 根据拟解决问题选取一种贪心准则；
(2) 按贪心准侧标准对 n 个候选输入排序（以这一方法为代表，仍可基于堆
来存储候选）。
